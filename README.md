# User Registration Form

This project is a simple **HTML User Registration Form** designed using basic HTML elements. It is suitable for beginners learning form controls and HTML structure.

## Features
- Full Name and Username input
- Email and Password fields with validation
- Gender selection using radio buttons
- Hobbies selection using checkboxes
- Date of Birth and Age input
- Country selection using dropdown list
- Submit and Reset buttons

## Technologies Used
- HTML5

## Purpose
This project helps beginners understand:
- HTML form elements
- Input types and attributes
- Basic form layout and structure
